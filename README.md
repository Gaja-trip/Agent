# 토지정보 통합 안내 페이지

토지이음, 다음지도, 국가법령정보센터를 탭으로 연결하고 토지 상담과 토지 정리 절차를 설명하는 공개용 정적 웹페이지입니다.

## 열기

`index.html` 파일을 브라우저에서 바로 열면 됩니다.

## 구성

- `index.html`: 메인 탭 화면
- `info.html`: 정보 연결 안내
- `process.html`: 토지 정리 진행 절차
- `checklist.html`: 상담 전 준비자료 체크리스트
- `styles.css`: 반응형 화면 스타일
- `script.js`: 메인 탭, 절차 탭, 준비자료 체크리스트 상호작용
