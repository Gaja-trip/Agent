# 토지정보 통합 안내 페이지

토지이음, 토지이음 지도, 국가법령정보센터를 메인 페이지 안에서 탭으로 연결하고 토지 상담과 토지 정리 절차를 설명하는 공개용 정적 웹페이지입니다.

## 열기

연속지적도 SHP 파일을 함께 불러오기 위해 로컬 서버로 열어야 합니다.

```bash
node server.cjs
```

실행 후 브라우저에서 `http://127.0.0.1:4173/`을 엽니다.

## 구성

- `index.html`: 메인 탭 화면
- `process.html`: 토지 정리 진행 절차
- `checklist.html`: 지적측량 안내자료와 상담 체크리스트
- `manual.html`: 통합 화면 사용 매뉴얼
- `knowledge.html`: 상담지식 요약
- `styles.css`: 반응형 화면 스타일
- `script.js`: 메인 탭, 절차 탭, 안내자료 출력, 체크리스트 상호작용
- `server.cjs`: 연속지적도 SHP를 함께 제공하는 로컬 정적 서버

## V-World 항공사진

메인 화면의 항공사진 탭은 토지이음 주소검색 값을 V-World 항공사진 위에 표시합니다.
주소를 검색하면 `cadastral` 폴더의 연속지적도 SHP를 읽어 주소 기준 50m 이내 필지 도형을 항공사진에 중첩합니다.
