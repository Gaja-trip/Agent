Manual

매뉴얼

토지이음, 토지이음 지도, V-World 항공사진, 법령정보를 같은 지번 기준으로 확인하는 순서입니다.

1

Office Start

사무실 출근 후 먼저 해야 할 일

출근하면 가장 먼저 퇴근한 사이 가상계좌로 수입이 되었는지 확인합니다. 전날 입금 내역을 확인한 뒤, 접수창 오른쪽 상단의 접수일을 실제 접수 기준일로 변경해 줍니다. 가상계좌 수입 화면을 누르면 조회 창이 열리며, 수입일자를 그 전날로 지정해 검색하면 입금내역을 확인할 수 있습니다.

  • 업무처리 화면에서 가상계좌 수입 금일 현황을 확인합니다.
  • 가상계좌 수입 창에서 수입일자를 전날로 맞춘 뒤 검색 버튼을 눌러 입금내역을 확인합니다.
  • 전날 입금이 확인되면 해당 건을 접수 처리하기 전에 접수일을 변경합니다.
  • 전날이 공휴일인데 입금이 되었으면, 그 전날이 아니라 다음날인 평일 일자로 접수합니다.

2. 지번 입력

메인 화면의 토지이음 기준 지번/주소 입력칸에 상담 대상 토지의 주소나 지번을 입력하고 적용합니다.

3. 지도 대조

토지이음과 토지이음 지도 탭에서 같은 지번이 표시되는지 확인하고, 용도지역과 주변 현황을 대조합니다.

4. 항공사진 확인

항공사진 탭에서 같은 지번의 실제 현장 형태와 접근 조건을 확인합니다.

5. 법령 근거 정리

국가법령정보센터에서 허가, 행위제한, 지적공부 정리에 필요한 근거 조항을 상담 메모에 남깁니다.
